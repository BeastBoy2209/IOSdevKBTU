### 4️⃣ Добавь первый пакет - Alamofire

```
┌──────────────────────────────────────────────────┐
│ Package Dependencies                             │
│                                                  │
│ Currently: 0 packages                            │
│                                                  │
│ ┌──────────────────────────────────────────────┐│
│ │ (пусто)                                      ││
│ └──────────────────────────────────────────────┘│
│                                                  │
│                            [+] [-] ← Кликни [+] │
└──────────────────────────────────────────────────┘
```

**Кликни кнопку [+]**

Появится окно:
```
┌──────────────────────────────────────────────────┐
│ Add Package Dependency                           │
├──────────────────────────────────────────────────┤
│ Search:                                          │
│ ┌──────────────────────────────────────────────┐│
│ │ https://github.com/Alamofire/Alamofire.git  ││ ← Вставь сюда
│ └──────────────────────────────────────────────┘│
│                                                  │
│ [Cancel]                    [Add Package]        │
└──────────────────────────────────────────────────┘
```

**Действия:**
1. Вставь URL: `https://github.com/Alamofire/Alamofire.git`
2. Нажми Enter или кликни поиск
3. Подожди 5-10 секунд
4. Выбери версию "Up to Next Major Version" (5.9.0)
5. Кликни **[Add Package]**

Появится второе окно:
```
┌──────────────────────────────────────────────────┐
│ Choose Package Products                          │
├──────────────────────────────────────────────────┤
│ ☑️ Alamofire                                     │ ← Убедись что выбрано
│                                                  │
│ Add to Target: Assignment10                     │
│                                                  │
│ [Cancel]                    [Add Package]        │
└──────────────────────────────────────────────────┘
```

**Кликни [Add Package]**

Подожди пока Xcode загрузит пакет (10-30 секунд).

---

### 5️⃣ Добавь второй пакет - Kingfisher

**Повтори то же самое:**

1. Снова кликни **[+]** в разделе Package Dependencies
2. Вставь: `https://github.com/onevcat/Kingfisher.git`
3. Нажми Enter
4. Выбери "Up to Next Major Version" (7.10.0)
5. Кликни **[Add Package]**
6. Убедись что **☑️ Kingfisher** выбран
7. Кликни **[Add Package]**

## 🎉 Что должно произойти:

1. Закрой симулятор или нажми Stop (⬛)
2. Запусти снова (⌘R)
3. **Тот же герой появится МГНОВЕННО!** ⚡
4. Нет загрузки = UserDefaults работает! ✅

## 🐛 Если что-то не так:

### Ошибка: "Cannot find 'AF' in scope"
- **Решение**: Alamofire не добавлен. Повтори шаг 4.

### Ошибка: "Cannot find 'kf' in scope"
- **Решение**: Kingfisher не добавлен. Повтори шаг 5.

### Пакеты не загружаются
- **Решение 1**: Проверь интернет
- **Решение 2**: **File → Packages → Reset Package Caches**
- **Решение 3**: Перезапусти Xcode

### Build Failed
- **Решение**: 
  1. Нажми ⌘⇧K (Clean Build Folder)
  2. Подожди 5 секунд
  3. Нажми ⌘B снова
